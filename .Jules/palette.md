## 2025-05-15 - [Skip to Content Pattern]
**Learning:** In this project, accessibility is a priority. Adding a "Skip to Content" link at the start of the layout targeting a `<main id="main-content">` wrapper is a required pattern for global accessibility.
**Action:** Always ensure a "Skip to Content" link is present in the layout and the main content is wrapped in a semantic `<main>` tag with a matching ID.

## 2025-05-15 - [Icon Button Accessibility]
**Learning:** Icon-only buttons must have descriptive `aria-label` attributes to be accessible to screen reader users. This includes mobile menu triggers, close buttons, and carousel navigation.
**Action:** Audit all icon-only buttons and add appropriate `aria-label` attributes.
