## 2026-02-10 - Enhancing Hero Background
**Learning:** Adding subtle background decorations like a dotted grid and animated blurred blobs can significantly improve the "premium" feel of a hero section without requiring complex assets or distracting the user.
**Action:** Use CSS radial-gradients for grids and framer-motion with blur filters for blobs to fill empty space elegantly.

## 2026-05-15 - [Authentication Transition]
**Learning:** Transitioning from a waitlist-only ecosystem to a full authentication flow requires a reusable navigation component to maintain consistency across landing, login, and signup pages. Visual continuity (e.g., using the same radial grid patterns and motion blobs) helps reinforce brand identity during sensitive user actions like signing up.
**Action:** Always extract navigation into a shared component when moving beyond a single-page application. Ensure background decorations are consistent across the entire user journey.
