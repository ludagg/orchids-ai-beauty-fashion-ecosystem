## 2026-02-10 - Enhancing Hero Background
**Learning:** Adding subtle background decorations like a dotted grid and animated blurred blobs can significantly improve the "premium" feel of a hero section without requiring complex assets or distracting the user.
**Action:** Use CSS radial-gradients for grids and framer-motion with blur filters for blobs to fill empty space elegantly.

## 2025-05-14 - Programmatic Motion Sequences
**Learning:** Programmatic motion sequences using Framer Motion provide a more performant and brand-consistent alternative to heavy MP4 videos or YouTube iframes for product demos. They allow for precise control over timing, interactions (like simulated cursors), and are perfectly smooth on GPU-accelerated devices.
**Action:** When asked for a video demo, propose or implement a programmatic React sequence if actual media files are not required, ensuring each scene is isolated and sequencing is managed via robust state/timers.
